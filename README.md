# Jogo da Forca

## Autor

Luiz Gabriel Abreu Oliveira

## Descrição

Este projeto consiste em um jogo da forca desenvolvido em JavaScript utilizando Node.js e executado diretamente pelo terminal.

O objetivo do jogo é descobrir a palavra secreta antes que o número máximo de tentativas seja atingido.

---

## Regras do Jogo

* O jogador deve tentar adivinhar a palavra secreta digitando letras.
* A cada letra incorreta, o jogador perde uma tentativa.
* O jogo exibe:

  * Letras já descobertas
  * Letras tentadas
  * Número de tentativas restantes
* O jogador vence quando descobre todas as letras da palavra.
* O jogador perde quando o número de tentativas chega a zero.

---

## Como Jogar

1. Execute o jogo pelo terminal.
2. Digite seu nome.
3. O jogo escolherá uma palavra aleatória.
4. Digite uma letra por vez para tentar descobrir a palavra.
5. Continue jogando até vencer ou perder.

## Como Executar

### Pré-requisitos

* Node.js instalado na máquina

### Executando o projeto

Abra o terminal na pasta do projeto e execute:

```bash
npm start
```
